Keyboard Control

> <

We aimed to develop a AL based Virtual Keyboard. It is necessary to develop a new type of environment-friendly virtual keyboard with fewer flaws. The proposed virtual keyboard can be put to use for a smartphone in the future. Furthermore, as a blueprint, it can be applied to computers after it is improved to allow ten-finger recognition. Moreover, machine learning can be potentially embedded into our virtual keyboard so as to greatly improve its finger recognition performance.
